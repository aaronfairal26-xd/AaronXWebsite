What is this hackathon about?

This is a college-level hackathon focused on web development and UI/UX. Build a working website demo and a live link. Demonstrate a clear user problem and your solution within the time limit.

  • Use any AI tools for design and coding.
  • Teams allowed (up to 5) or individual participation.
  • Deliver a live demo link (GitHub Pages/Netlify) and a short presentation.

Rules, Judging & Schedule

Everything teams need to know: allowed tools, judging criteria, schedule, and submission details.

Allowed tools

You may use ChatGPT, GitHub Copilot, DALL·E and other AI tools for design and coding. Cite any external libraries or major assets in your submission.

Team & Time

Teams of up to 5 or individuals. Build a working website and provide a live link. Time limit: 8 hours on hackathon day.

Judging Criteria
  • User experience & design (30%)
  • Functionality & demo (30%)
  • Innovation & usefulness (20%)
  • Presentation & code cleanliness (20%)
Submission

Submit a live link (GitHub Pages / Netlify) and a link to the repository. Include a short README explaining setup and who did what.

Tips for designers
  1. Keep the scope small and the UI uncluttered.
  2. Prepare assets beforehand (icons, hero images).
  3. Test mobile layout often.
